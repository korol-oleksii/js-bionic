Завдання:
        Доопрацюйте сценарій. Якщо користувач ввів значення 0 або не числове значення для висоти та радіуса циліндра -
        повідомте про помилку.
        Якщо введені значення коректні – розрахуйте обсяг циліндру.
        Якщо конвертація рядкового значення до цілочисельного пройшла не успішно, то результат конвертації NaN.
        Для перевірок значення в умові на рівні з NaN потрібно використовувати функцію isNaN(value),
        яка поверне true, якщо значення value не є числом.
        Така перевірка – value == NaN, не спрацює.
